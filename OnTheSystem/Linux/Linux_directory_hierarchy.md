# Linux directory hierarchy

算是 linux 必知必会的一个小知识点吧。因为linux 的包管理器确实做了太多事情，导致有时候甚至不知道安装好的包放在哪里。

| Directory         |      |      |
| ----------------- | ---- | ---- |
| `/bin`            |      |      |
| `/sbin`           |      |      |
| `/usr/bin`        |      |      |
| `/usr/sbin`       |      |      |
| `/usr/local/bin`  |      |      |
| `/usr/local/sbin` |      |      |
